Fetch the current weather and forecast for today using a weather API (OpenWeatherMap or similar). Auto-detect the user's location based on their IP address.

Return a nicely formatted weather notification with:
- Current temperature and weather conditions
- High/low temperatures for the day
- Brief summary of the forecast
- Any weather alerts if applicable

Format the response as a clear, readable notification that the user can quickly scan in the morning. Keep it concise but informative.
